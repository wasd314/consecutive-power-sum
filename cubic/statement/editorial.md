# 解説

まず，解の候補を絞りましょう．$(L, R)$ が解であるとき，
$$
R^3 \le \sum_{i = L}^R i^3 = S(L, R) = N
$$
より $R \le N^{1/3}$ です．よって $M \coloneqq \lfloor N^{1/3} \rfloor$ とおき，数列 $A = (A_1, A_2, \dots, A_M)$ を $A_i \coloneqq i^3$ で定めると，本問題は「数列 $A$ の区間和が $N$ になる区間を列挙せよ」と言い換えることができます．以下ではこの言い換えた問題の解法を2通り紹介します．


# 方針1: 尺取り法

数列 $A$ の各項は正整数なので，区間和は区間幅に対して単調性が成り立ちます．より正確には，区間の始点を固定して終点を伸ばしていくにつれて，対応する区間和は単調に大きくなっていきます．このような単調性が成り立つ状況では尺取り法を用いることができます．

本問題では区間に対応する区間和 $s$ について $s \le N$ を保つようにすると，全体 $\Theta(M)$ 時間で解を列挙できます．また区間を広げるかの判定の際にも高々 $s + A_i \le N + N = 2 N < 2^{61}$ の値を扱えれば良く，64 bit 整数型を使用すれば十分です．


# 方針2: 二分探索

数列の区間和に対しては，数列の累積和を考えることも有効です．数列 $B$ を
- $B_0 \coloneqq 0$
- $B_{i} \coloneqq B_{i - 1} + A_i \quad (i = 1, 2, \dots, M)$

もしくは直接
$$
B_{k} \coloneqq \sum_{i=1}^{k} A_{i} = \sum_{i=1}^{k} i^3 = \frac{1}{4} k^2 (k+1)^2
$$
で定めると $S(L, R) = B_R - B_{L-1}$ が成り立つので，区間和の条件は
$$
B_R - B_{L-1} = N
$$
と表せます．

$L$ を $M$ 以下で全探索することを考えます．数列 $B$ が狭義単調増加することから，各 $L$ に対して $B_R = B_{L-1} + N$ なる $R$ は高々1つであり，そのような $R$ は $B_i$ から $i$ を逆引きする連想配列を用意しておくことで $\Theta(1)$ 時間もしくは $\Theta(\log M)$ 時間などで求められます．よって全体 $\Theta(M)$ 時間もしくは $\Theta(M \log M)$ 時間などで解を列挙できます．

なおこの方針では $B_M \sim \frac{1}{4} N^{4/3}$ は 64 bit 整数型に収まらないことがあるため，オーバーフローに注意が必要です．


# まとめ

以上のようにして本問題を全体 $\Theta(M) = \Theta(N^{1/3})$ 時間もしくは $\Theta(M \log M) = \Theta(N^{1/3} \log N)$ 時間で解くことができます．
